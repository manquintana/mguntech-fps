# FPS: FBI Pistol Standalone 

FPS (FBI Pistol Standalone) is a mobile shooting simulator designed to replicate FBI-style target practice without the need for automated targets.

The application allows users to train and practice shooting scenarios in a controlled and accessible digital environment.

Note: This app is no longer maintained in Google Playstore, but feel free to build your own. All required source code is in here :)
